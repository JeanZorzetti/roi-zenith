### 4.2 Form Validation & States

**Form States:**
- Default: Empty fields with placeholder text
- Filled: User typing, border color changes
- Valid: Green checkmark icon
- Invalid: Red border + error message below field
- Submitting: Button shows loading spinner
- Success: Form replaced with success message
- Error: Error banner above form

**Validation Rules:**
```typescript
interface ContactFormData {
  name: string;        // Required, min 2 chars
  email: string;       // Required, valid email format
  company?: string;    // Optional
  interest: string;    // Required, dropdown selection
  message: string;     // Required, min 10 chars
  consent: boolean;    // Required, must be true
}
```
